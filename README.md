# Fishcake RWS Presentation

## About

This is the official presentation deck for Fishcake - The RWS Standard Layer for real-world Web3 interactions.

**AI x Web3 Shenzhen Hackathon | Jan 24-25, 2026**

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```
